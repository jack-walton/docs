# KaTeX Support

This theme includes built-in KaTeX support for rendering mathematical expressions in your content. See the [KaTeX documentation](https://katex.org/docs/supported.html) for more information.

## Examples

Wrap a mathematical expression in the dollar sign (`$`) to enable KaTeX format like this: `$E = mc^2$`. For multiple lines of math, use two dollar signs (`$$`) to define the length of the math block.

```
$$
\begin{equation}
\sum_{i=1}^{k+1} i = \left(\sum_{i=1}^{k} i\right) + (k+1)
\end{equation}
$$
```
