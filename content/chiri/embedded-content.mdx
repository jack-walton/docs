# Embedded Content

import {Callout} from 'nextra/components'

Use these shortcodes to embed media:

- `::link{url="https://xxxxx.xxx"}`
- `::spotify{url="https://open.spotify.com/type/xxxxx"}`
- `::youtube{url="https://www.youtube.com/watch?v=xxxxx"}`
- `::bilibili{url="https://www.bilibili.com/video/xxxxx"}`
- `::github{repo="username/repo"}`
- `::x{url="https://x.com/username/status/xxxxx"}`
- `::neodb{url="https://neodb.social/category/xxxxx"}`

<Callout type="info">

When embedded content is still loading, the table of contents positioning may be inaccurate.

  </Callout>

## Link Card

This shortcode is an example of how to embed a link card in the content:

```
::link{url="https://pitchfork.com/reviews/albums/ichiko-aoba-luminescent-creatures/"}
```

<Callout type="info">

To use the link card short code, you need to set up an adapter in `src/astro.config.ts`. If you don’t need the link card shortcode, you can set `linkCard` to `false` in `src/config.ts`. In that case, you don’t need to configure the adapter before building.

  </Callout>
